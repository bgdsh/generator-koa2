> Scaffold out a koa2 app

## Install

```sh
$ npm install -g generator-koa2
```

> Note: `generator-koa2` as a [yeoman](http://yeoman.io/) generator, so you must have global `yo` installed, too.

```sh
$ npm install -g yo
```

## Usage

```sh
$ yo koa2
```
